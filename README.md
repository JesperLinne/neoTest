This project implements a password generator using React, TypeScript and the Neo4j Needle component library.

## Features

- Custom accessible slider (keyboard + mouse + pointer drag)
- Password generation based on selected character groups
- Explicit password generation (updates only when pressing "Generate")
- Copy to clipboard (implemented via Needle's useCopyToClipboard hook)
- Keyboard accessible UI
- Automated tests (Slider, checkbox interactions, password generation)
- Language translations using react-i18next

## Tech Stack

- React + TypeScript
- Vite
- Neo4j NDL components
- Vitest + Testing Library
- React-i18next (i18n)

## Getting Started

npm install
npm run dev

## Testing

npm run test
